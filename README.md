### **How to interact with the contract**

#### Prerequisite:
  Install Metamask browser plugin. Link can be found [here](https://metamask.io/download).
  Open Metamask account. For the sake of interact with out applicaton please use **Kovan Testnet** and use [Faucet](https://linkfaucet.protofire.io/kovan) to get some test ETH
  1. Clone the repository.
  2. Open prefferable terminal in the root directory and type:
     >pip install -r requirements.txt
  3. Run main.py
  4. Type in terminal:
     > uvicorn main:app --reload
  6. Enter a browser and press: http://127.0.0.1:8000/
  7. **Hooray**
